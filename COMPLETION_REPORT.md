## 🚀 使用方式

### 1. 立即查看

访问以下URL查看效果:
- 标签页: `http://你的域名/tags/`
- 分类页: `http://你的域名/categories/`

### 2. 快速自定义

编辑 `assets/css/extended/bubble-terms.css`:

```css
/* 改变泡泡大小 */
.bubble-item[data-count="1"] { --item-size: 100px; }

/* 改变动画速度 */
animation: float 5s ease-in-out infinite;

/* 改变透明度 */
background: rgba(255, 255, 255, 0.25);

/* 改变模糊度 */
backdrop-filter: blur(20px);
```

### 3. 高级配置

查看以下文件获取更多选项:
- `BUBBLE_CONFIG_EXAMPLES.css` - 预设配置
- `BUBBLE_FEATURE_DOCS.md` - 完整文档

## 🔧 维护和更新

### 常见维护任务

1. **改变泡泡大小**
   - 编辑: `assets/css/extended/bubble-terms.css`
   - 修改: `.bubble-item[data-count="X"]` 的 `--item-size` 值

2. **改变动画速度**
   - 编辑: `assets/css/extended/bubble-terms.css`
   - 修改: `.bubble-item` 的 `animation` 时间

3. **改变颜色/透明度**
   - 编辑: `assets/css/extended/bubble-terms.css`
   - 修改: `background` 和 `backdrop-filter` 值

4. **仅在标签页启用**
   - 编辑: `layouts/partials/extend_head.html`
